# Personal Diary Tracker Bot

Telegram-бот для трекинга сна, чек-инов, еды/напитков, заметок, напоминаний, экспорта и конвертации файлов.

## Возможности
- 🛌 Сон (1 запись в день)
- ⚡️ Чек-ины (энергия/стресс/эмоции)
- 🍽🥤 Еда и напитки
- 📝 Заметки и напоминания
- 📊 Статистика
- 📤 Экспорт данных и загрузка медиа через `yt-dlp`
- 🔄 Конвертация файлов через `ffmpeg`
- 🌍 Учет локального часового пояса пользователя

## Требования
- Python 3.10+
- Telegram Bot Token
- `ffmpeg` в PATH (или бинарник `./ffmpeg` рядом с проектом)

## Установка
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
